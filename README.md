   Questão 1

O HeapSort sempre tem complexidade O(n log n) porque trabalha com uma árvore binária (Heap), e cada remoção leva log n.

O ShellSort não tem complexidade fixa porque depende da sequência de gaps utilizada, podendo variar o desempenho.
